# Super Pest Control Mumbai

A modern, SEO-optimized, and fully accessible website for Super Pest Control Mumbai - professional pest control services across Mumbai with licensed, eco-friendly solutions.

## 🌟 Features

- **SEO Optimized**: Complete meta tags, Schema.org markup, sitemap, and robots.txt
- **Fully Accessible**: WCAG 2.1 AA compliant with screen reader support
- **Responsive Design**: Mobile-first approach with smooth animations
- **Performance Optimized**: Fast loading times, optimized assets
- **Modern Tech Stack**: HTML5, Tailwind CSS, Vanilla JavaScript
- **PWA Ready**: Progressive Web App capabilities with manifest.json

## 🚀 Performance Targets

- ✅ First Contentful Paint: <2 seconds
- ✅ Largest Contentful Paint: <3 seconds
- ✅ Total page weight: <500KB
- ✅ Lighthouse Score: 95+ (Performance, SEO, Accessibility)

## 📱 Responsive Breakpoints

- **Mobile**: 320px - 640px
- **Tablet**: 641px - 768px
- **Desktop**: 769px - 1024px
- **Large**: 1025px+

## 🔧 Development

### Local Development
1. Clone or download the project
2. Open `index.html` in a modern web browser
3. For live reloading, use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```
